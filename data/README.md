# Data Directory

This directory contains dynamic data generated by the AusSuperPredictor application:

## Files (Generated automatically - not in version control)

- `australian_super_daily.csv` - Market data updated when you click "Update Data"
- `model.pkl` - Trained machine learning model (generated when you click "Train Model") 
- `features.pkl` - Feature definitions used by the model

## First Run Setup

1. Click "Update Data" to download initial market data
2. Click "Train Model" to create your first model
3. Click "Run Prediction" to get next-day predictions

All data files are automatically created when needed.
